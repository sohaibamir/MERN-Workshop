To install react router for smooth navigation of pages:
npm i react-router-dom

--> We have used material ui for styling of multiple components.
To install mui package in react:
npm install @mui/material @emotion/react @emotion/styled

MUI Official Website:
https://mui.com/material-ui


To use react icons in the project:
https://react-icons.github.io/react-icons/

1 --> npm i react-icons
2 --> select the icon you want to use from the Website
3 --> import the icon
4 --> incorporate in your project
